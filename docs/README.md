# Documentatie

Acest folder contine documentatia tehnica locala a proiectului.

Fisiere:
- `betonquest-directii-potrivite-pentru-ainpc.md` - analiza aplicata a caracteristicilor BetonQuest care merita preluate in AINPC si a celor care nu trebuie copiate direct
- `documentatie-api.md` - contractul public curent pentru `ainpc-api` si regulile de integrare pentru addonuri
- `generare-ai-si-constructie-automata.md` - directie pentru generare de sate, castele, pesteri si constructie automata cu recomandarea `WorldEdit API` + orchestration proprie
- `implementat-deja.md` - rezumat al functionalitatii existente deja in cod
- `mapping.md` - design pentru sistemul de mapare si etichetare a locatiilor
- `npc-uri-temporare-si-episodice.md` - design pentru NPC-uri temporare, low-impact si non-villager
- `questuri-avansate.md` - documentatie pentru evolutia sistemului de questuri la nivel avansat
- `reactie-npc-jucator.md` - design pentru sistemul de reactie dintre NPC si jucator
- `roadmap-orientativ.md` - roadmap pe faze pentru release, modularizare, scenarii extensibile si AI-assisted authoring
- `strategie-plugin-modular-si-scenarii-programabile.md` - strategie pentru arhitectura modulara si scenarii extensibile

Observatii:
- `TODO.md` ramane in radacina proiectului ca lista de lucru
- `docs.text` ramane separat ca document intern de referinta
